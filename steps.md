# Streamlit HTML Deployment Demo

## 1. Create project folder and files
Create a folder named:
streamlit-html-demo



### requirements.txt
streamlit


--------------------------------------------------

## 2. Install Streamlit (optional for local run)
pip install streamlit

## 3. Run locally (optional)
streamlit run app.py


--------------------------------------------------

## 4. Initialize Git
git init

## 5. Add files
git add .

## 6. Commit
git commit -m "first commit"


--------------------------------------------------

## 7. Create GitHub Repository
Go to https://github.com and create a new repository named:
streamlit-html-demo

DO NOT add README while creating


--------------------------------------------------

## 8. Connect and Push to GitHub
git remote add origin https://github.com/YOUR-USERNAME/streamlit-html-demo.git
git branch -M main
git push -u origin main


--------------------------------------------------

## 9. Deploy on Streamlit Cloud

1. Go to: https://streamlit.io/cloud
2. Click "New App"
3. Select your GitHub repo
4. Select branch: main
5. Select file: app.py
6. Click Deploy


--------------------------------------------------

## 10. Output

You will get a live URL like:
https://your-app-name.streamlit.app

Open it in browser to see your website running 


--------------------------------------------------

## Final Flow

Create files → Git init → Commit → Push → Deploy → Get URL
